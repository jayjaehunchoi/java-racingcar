# TDD 덧셈 계산기 요구사항

1. 커스텀 구분자는 문자열 앞부분의 “//”와 “\n” 사이에 위치하는 문자를 커스텀 구분자로 사용한다.
    * //가 있는지 확인한다.
    * 정규식 **"[/]{2}.\\s"** 으로 검증한다 (커스텀 구분자는 한글자만 허용)
    * 구분자가 숫자일 경우 예외 처리한다.

2. 커스텀 구분자가 없을 경우, 쉼표(,) 또는 콜론(:)을 구분자로 사용한다.
    * split을 , 와 : 로 한다.
    * (예외) 배열을 정수형으로 변경하는 과정에서 에러를 발생시킨다.
    

3. 숫자가 아닐때와 0보다 작은 경우 정수형 배열로 변경 후 예외 발생시킨다.
